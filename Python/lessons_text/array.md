# 1. Списки в Python
Список (list) — это упорядоченные элементы. Можно хранить числа, строки и любые объекты.

В списки, как и в переменных, данные можно приписывать по разному

```python
a = [5, 2, 10, 1]
b = []               # пустой список
n = int(input())
arr = [int(input()) for _ in range(n)]

print(a)
print(b)
print(arr)
```
Ответ программы: 

[5, 2, 10, 1]

[]

[1, -5, 0, 15, 28]

Сама особенность списков заключается в том, что у каждого элемента есть своё назначенное место, от 0 до размера списка.

Так в списке a = [5,2,10,1], 5 имеет место 0, 2 имеет место 1 и тд. Само "место", прописывается в квадратных скобках

```python
a = [5, 2, 10, 1]
print (a[0])  # вывод первого элемента списка(5)
print (a[2])  # вывод третьего элемента списка(10)
```

Ответ программы:

5

10

Эта особенность списков, создает небольшую особенность как мы взаимодействуем с элементами внутри него, если мы хотим создать список, и ввести внутрь него цифры, мы не можем просто написать:

```python
a[5] = int(input())
```

Ответ программы:
```
IndexError                                Traceback (most recent call last)
Cell In[17], line 1
----> 1 a[5] = int(input())

IndexError: list assignment index out of range
```

Как видно, выходит ошибка, вместо этого нам надо использовать штуку которую мы проходили до этого, in range:

```python
a = [int(input()) for _ in range(5)] # создание списка с вводом 5 элементов
print (a)
```

Ответ программы: 

[5, 3, 2, -4, 1]

Для вывода списка на экран, in range использовать не надо, просто пишем print(a)

# 2. Действия с списками
У списков есть несколько различных операций которые мы можем с ними выполнить, а именно:
1. Длина списка: len

```python
a = [5, 2, 10, 1]
print (len(a))  # длина списка
```
Ответ программы: 

4

2. Добавить элемент к уже существующему списку: append
```python
a = [5, 2, 10, 1]
a.append(7) # добавляем элемент 7 в конец списка
print(a)
```

Ответ программы:

[5, 2, 10, 1, 7]

3. Сортировать список по возростанию: sort
```python
a = [5, 2, 10, 1]
a.sort()  # сортируем список по возрастанию
print(a)
```

Ответ программы:

[1, 2, 5, 10]

4. Сортировка по убыванию
```python
a = [5, 2, 10, 1]
a.sort(reverse=True)  # сортируем список по убыванию
print(a)  
```

Ответ программы:

[10, 5, 2, 1]
